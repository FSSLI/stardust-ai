# 用户画像

## 核心用户群体

### 1. 都市独居青年（18-30岁）
- **特征**：独自在大城市生活，社交圈小，工作压力大
- **痛点**：孤独感、无人倾诉、情绪无处释放
- **需求**：随时可聊天的伙伴、情感支持、倾听者

### 2. 学生群体（16-25岁）
- **特征**：学业压力大，对未来迷茫，社交焦虑
- **痛点**：不好意思向朋友倾诉、害怕被评判
- **需求**：无压力的倾诉对象、学习陪伴、情绪调节

### 3. 情感需求者（22-35岁）
- **特征**：渴望被理解、被关心，但现实中难以获得
- **痛点**：人际关系复杂、害怕麻烦别人
- **需求**：专属陪伴、记住自己的喜好和经历、稳定的存在

## 用户场景

| 场景 | 描述 |
|------|------|
| 深夜emo | 凌晨睡不着，想找人说说话，星尘在线陪伴 |
| 日常分享 | 今天吃了好吃的、遇到了有趣的事，随手分享 |
| 情绪低谷 | 工作受挫、感情不顺，需要一个倾听者 |
| 记录生活 | 让星尘帮忙记录重要事项，形成手帐 |
| 角色扮演 | 切换不同人格，体验不同的陪伴风格 |

## 用户核心诉求

1. **被记住**：不要每次对话都像第一次认识
2. **被理解**：能感知情绪，给出合适的回应
3. **被陪伴**：随时在线，稳定存在
4. **有趣**：不只是问答，有性格、有温度
